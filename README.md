## USER STORY

- 按下按鈕或Enter鍵添加任務
- 使用標籤將任務分類，選單切換分類頁面
- 可以記錄上次用過的標籤
- 在列表中刪除指定任務，也可以一次刪除各分類中全部任務
- 點擊項目表示打勾完成該任務（再按一下可以取消）
